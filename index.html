About

This is a frontend-only GitHub Pages application designed for AI-native solo developers who want to track activity, issues, and PRs across all their projects in one place.

The dashboard will provide:

  • Unified view of multiple repositories
  • Activity tracking and commit history
  • Issue and PR monitoring
  • Organization support
  • Personal notes and tags
