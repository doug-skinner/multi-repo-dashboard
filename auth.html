Welcome to Multi-Repo Dashboard

A unified dashboard for tracking your GitHub repositories

Getting Started

To use this dashboard, you'll need to provide a GitHub Personal Access Token (PAT). This token allows the dashboard to access your repositories and activity data.

What you can do:

  • Track multiple repositories in one place
  • Monitor issues, pull requests, and commits
  • View activity across personal and organization repos
  • Customize your dashboard with filters and tags

Setup Your Access Token

Create a GitHub Personal Access Token:

  1. Go to GitHub Token Settings
  2. Give your token a descriptive name (e.g., "Multi-Repo Dashboard")
  3. Select the following scopes:

Required Scopes:

  • repo - Full control of private repositories
  • read:user - Read user profile data
  • read:org - Read organization data
